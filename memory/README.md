# AutoHarness Memory

本目录存储 AutoHarness 的跨会话学习记忆。

## 文件结构

- `global_principles.md`: 全局原则（跨模板类型通用，≤20 条）
- `templates/`: 各模板类型的专属知识（每类 ≤30 条）

## 原则

1. 优先使用 guard clause 减少嵌套
2. 避免在 match 分支中执行复杂逻辑
3. 保持函数短小（< 50 行）
4. 验证优先于修复
5. 状态转换需完整覆盖

## 使用方式

1. **自动写入**：每次合成结束后自动更新
2. **手动编辑**：可直接修改 Markdown 文件
3. **版本控制**：所有更改通过 Git 管理

## 清理策略

- 全局原则达到 20 条时，按频率衰减淘汰最低频原则
- 模板知识达到 30 条时，合并相似模式，保留最高频

## 备份机制

每次写入自动创建 `.md.bak` 备份，可通过 `rollback()` 回退。
